## 🚀 Fitur Utama
1. **State Management (Provider)**: Mengelola state counter secara terpusat dan efisien tanpa membangun ulang (rebuild) seluruh tampilan halaman.
2. **Local Notifications**: Mengirimkan banner notifikasi instan secara lokal ke sistem Android (`Heads-up Notification`) setiap kali tombol penambah counter diklik.
3. **Premium UI/UX Design**: Antarmuka modern bertema luar angkasa gelap (*deep space*) menggunakan gradasi warna harmonis dan kartu identitas personal.

## 🛠️ Cara Menjalankan Proyek

### 1. Prasyarat
- Flutter SDK terinstal (versi terbaru disarankan).
- Android SDK terinstal.
- Koneksi ke emulator Android atau perangkat Android fisik.

### 2. Langkah Penginstalan
Jalankan perintah berikut di dalam direktori `tugas-13/source code/`:

```bash
# Bersihkan cache build
flutter clean

# Unduh dependensi
flutter pub get

# Jalankan aplikasi ke perangkat target
flutter run
```
